# Command syntax #

{% apibody %}
array.changeAt(index, value) &rarr; array
{% endapibody %}

# Description #

Change a value in an array at a given index. Returns the modified array.

__Example:__ Bruce Banner hulks out.

```java
r.expr(r.array("Iron Man", "Bruce", "Spider-Man")).changeAt(1, "Hulk")
 .run(conn);
```
